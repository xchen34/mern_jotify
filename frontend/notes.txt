frontend setup:
Vite 是一个前端「构建工具 / 开发服务器（build tool + dev server）」 Vite = 帮你“跑”和“打包”前端项目的工具
 Vite 主要做 3 件事：

1️⃣ 本地开发服务器
npm run dev
浏览器秒开
热更新（HMR）
2️⃣ 模块打包
把很多文件打包成生产版本
用的是 Rollup（底层）
3️⃣ 现代 JS 支持
ES modules
JSX / TS
自动处理依赖
 
 npm create vite@latest .  
 选择react, javascript 
如果没有自动启动就  npm run dev 
就可以访问 http://localhost:5173/  它给的网站

 若node版本过低(node -v 查看)  
 nvm install 20

 使生效  source ~/.zshrc
